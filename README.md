# 💱 Currency Converter

A real-time currency converter web application built using HTML, CSS, and JavaScript. It allows users to convert currencies dynamically using live exchange rates fetched from an external API.

## 🚀 Features

* 🌍 Convert currencies in real-time
* 🔄 Live exchange rates using API
* 🏳️ Dynamic country flags update
* 🎯 Default selection (USD → INR)
* ⚡ Instant result display
* 🎨 Clean and responsive UI

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* REST API (Exchange Rate API)

## ⚙️ How It Works

* User enters amount
* Selects “From” and “To” currencies
* JavaScript fetches real-time rates from API
* Calculates converted value
* Updates result dynamically on UI

## 🔗 API Used

* Exchange Rate API
https://open.er-api.com/v6/latest
